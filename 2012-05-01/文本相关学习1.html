nano （以前用 现在不用）

  • 文本编辑处理 可能有的发行版本没有 yum install 以下即可

vi | vim

  • vim Vi IMproved 理解为升级或者 vi 加强

  • 配置相关

  • 编辑模式 等待编辑命令输入

  • 插入模式 一般输入 i 写入文本信息 按ESC 切换到编辑模式
  • 命令模式 一般输入 ： wq q! x
  • 编辑模式 ctrl+f 下一页 ctrl+b 上一页
  • g 首行
  • G 末尾行
  • ^行首字符
  • $行末字符
  • dd ndd 删除
  • uu 撤销删除
  • yy nyy 复制
  • p 粘贴
  • /searchword //向下搜
  • ?searchword //向上搜
  • :s/word/replace/g //替换
  • 命令模式 :set number

grep # global search regular expression(RE) and print out the line

  • 是一种强大的文本搜索工具，它能使用正则表达式搜索文本，并把匹配的行打印出来
  • 常常配合管道命令使用 cat /root |grep *.log
  • 匹配某个单词 grep word file1 (fileN) –color=auto
  • 除匹配外显示 grep -v “pattern” file

  • grep -E egrep 等价

  • grep -o 只显示匹配部分而不是全部展示匹配行
    • grep -A number 匹配文本之后的显示行数
    • grep -B number 匹配文本之前的显示行数
    • grep -c “pattern” file 统计匹配的行数
    • grep -C 显示匹配前后行数

  • grep -e “p1” -e “p2”

  • grep -q 安静模式 用于测试返回值是否为0

  • grep -r –include *.log
  • grep -r –exclude “*.php”
  • grep -r –exclude-from exclude-file-list
  • grep “*.log” /home/www -r -n

cut

  • 显示行中的指定部分(常用) 或者 删除文件中指定字段

  • 查看系统用户个数 cut -f 1 passwd -d “:” | wc -l
