文本相关学习2

May 10 2012

head

  • 显示文件的开头的内容
  • head -n 10 file //显示文件开始10行
  • head -c 10 file //显示文件开始10个字符

    tail

  • 显示文件末内容 与head相反
  • 常用命令 tail -f file 监事某个文件是否变更

    more

  • 分屏展示文件命令,支持向前翻页
  • more -c -5 passwd
  • more -dc passwd

less

  • 向前向后分屏查看文件内容 同more
  • less -N file

sed (stream editor)

  • 流编辑器
  • 常和管道命令搭配使用
  • 功能强悍 单独行匹配，在临时缓存处理内容，处理完成后，输出到屏幕

  • 替换操作 s/ 与贪婪模式 /g sed ‘s/search/replace/g’

  • sed -i 将在编辑文件字后，重新覆盖原文件

  • cat f1.log |sed ‘/^$/d’ //删除f1.log中的空白行
  • sed ‘5,$d’ -i f1.log //删除文件从5行开始至末尾
  • sed -n ‘5,$p’ f1.log //显示文件从5行开始至末尾

  • 匹配标记 &

  • sed -e ‘pattern1’ -e ‘pattern2’ file 多模式处理

wc

  • word count 单词计数

  • wc -l file //常和管道配合使用

awk 文本处理的瑞士军刀 ,来源于纪念三个人的首字母命名

  • awk [options] ‘BEGIN{ print “start” } pattern{ commands } END{ print “end” }’ file

  • 内置变量
    [A][N][P][G]表示第一个支持变量的工具，[A]=awk、[N]=nawk、[P]=POSIXawk、[G]=gawk

  • awk -F : ‘BEGIN{ print”Start” } { print $1} END{print”End”}’ mypas
