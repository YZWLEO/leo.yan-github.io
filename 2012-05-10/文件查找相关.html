文件相关

May 10 2012

which

  • 用于查找并显示给定命令的绝对路径
  • which –help
  • man which
  • info which

whereis

  • 命令用来定位指令的二进制程序、源代码文件和man手册页等相关文件的路径
  • 基于linux记录的数据库文件搜索（不实时）# updatedb

find

  • 根据选项参数查找指定目录下文件
  • 功能强悍，磁盘全扫描，效率相对低
  • find –help

strings 不常用

  • 在对象文件或二进制文件中查找可打印的字符串

diff

  • 比较给定的两个文件的不同
  • diff –help 功能强悍
  • diff 对比模式 默认模式|context模式 diff -c| Unified diff -u
  • diff -u 较常用 返回合并之后的文件 简洁明了

cmp （我不常用）

  • 二进制比较命令
  • 比较两个文件的内容并报告不同的第一个字符
  • cmp -[l|s] file1 file2
